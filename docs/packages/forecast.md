# forecast

[Back to the implemented-package index](../../README.md#implemented-packages).

## Overview

`forecast.f90` translates the
[forecast](https://cran.r-project.org/web/packages/forecast/index.html) R
package.

## Algorithms and Procedures

It contains benchmark forecasts, exponential smoothing, Croston and Theta
methods, transforms, Fourier regressors, correlations, accuracy measures, and
the Diebold-Mariano statistic.
