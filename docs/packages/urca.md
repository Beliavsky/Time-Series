# urca

[Back to the implemented-package index](../../README.md#implemented-packages).

## Overview

`urca.f90` translates the
[urca](https://cran.r-project.org/web/packages/urca/index.html) R package.

## Algorithms and Procedures

It contains ADF, KPSS, Phillips-Perron, ERS/DF-GLS, Zivot-Andrews, and Johansen
cointegration estimation.

Johansen supports deterministic terms, long-run/transitory specifications,
seasonal dummies, and external regressors.
