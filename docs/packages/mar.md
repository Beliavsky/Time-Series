# mAr

[Back to the implemented-package index](../../README.md#implemented-packages).

## Overview

`mar.f90` translates the
[mAr](https://cran.r-project.org/web/packages/mAr/index.html) R package.

## Algorithms and Procedures

It provides its augmented and column-scaled QR estimator for multivariate
autoregressions, phase-normalized companion eigenmodes with period and damping
diagnostics, PCA-subspace fitting, and stationary-mean Gaussian VAR simulation.
It reuses the shared linear algebra and random-number modules.
